# ReadMe

## Files

### Ex 3

All the files for Ex 3are in the `ex3` directory.

`ex3.sh` is code for the first part, and `ex3_r.sh` is the revised code for the second part.

### Ex 4

The file for Ex 4 is `ex4.c`.

## How to use

### Ex 3

Run:

```shell
./ex3.sh &
./ex3.sh
```

Or (for part 2):

```shell
./ex3_r.sh &
./ex3_r.sh
```

### Ex 4

To compile, use:

```
gcc -pthread -o ex4 ex4.c
```

And run:

```
./ex4
```
